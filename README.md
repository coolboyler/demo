# D-6 负荷预测展示 Demo

这个项目保留了原有网页风格，但核心数据已经切换为项目内置的 D-6 负荷预测流程，重点展示：

- 从 `2026-01` 开始的历史准确率
- `D-6` 主模型与 `0.5*D-7 + 0.3*D-14 + 0.2*D-21` 固定系数基线的对比
- 当前正式 `D` 日的分时预测曲线
- 录入新的 `D-6` 实际分时后，自动更新历史并生成下一次 `D` 日预测

## 项目结构

- `app/`: FastAPI 服务和页面接口
- `templates/`, `static/`: 页面模板与前端交互
- `pipeline/`: 已放入项目内的完整 D-6 预测脚本、历史数据和结果文件
  - `pipeline/scripts/`: 预测、更新、节假日路由等脚本
  - `pipeline/new/`: 历史日表、历史分时表、基线数据集、实际模板
  - `pipeline/results/`: 模型摘要、历史结果和当前预测产物

## 启动

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python run.py
```

启动后访问 `http://127.0.0.1:8008/`。

## 页面能力

### 1. 历史准确率展示

页面默认展示当前正式 `D` 日；也可以按日期切换查看历史目标日。

- 历史样本从 `2026-01-01` 开始
- 同时展示 `D-6` 模型和固定系数基线的日准确率、月准确率和分时对比
- 历史月内列表按目标日聚合展示，便于直接比较

### 2. D-6 实际录入

页面左侧提供“录入 D-6 实际分时”表单。

- 支持 Excel 一行/一列粘贴 24 个值
- 录入后会更新：
  - `pipeline/new/history_daily.csv`
  - `pipeline/new/history_hourly.csv`
- 同时自动生成下一次正式 `D` 日预测，并写入 `pipeline/results/forecast_d6_YYYYMMDD.*`

### 3. 固定系数基线

固定系数口径为：

```text
0.5 * D-7 + 0.3 * D-14 + 0.2 * D-21
```

页面对每个目标日同时展示：

- D-6 模型日电量
- 固定系数日电量
- 两种方法对应的准确率 / MAE / RMSE / 5% 命中
